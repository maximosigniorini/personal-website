# Maximo Signiorini | Audio & Tech Portfolio

[**🔴 Live Demo**]([https://www.maximosigniorini.com/])

## 📖 About This Project
This repository hosts the source code for my personal professional portfolio. As a **Sound Designer** pivoting into **Software Engineering**, I designed this platform to showcase my audio work while demonstrating my technical competency in modern web development.

The application serves as a central hub for my compositions, sound design reels, and creative coding projects.

## 🚀 Key Features
* **Custom Audio Players:** Implemented custom-styled media controls to ensure high-fidelity playback across browsers.
* **Responsive Design:** Fully fluid layout optimized for mobile, tablet, and desktop viewports.
* **Performance:** Optimized asset loading for heavy media files (audio/video) to maintain high Lighthouse performance scores.
* **Clean UI/UX:** Minimalist aesthetic designed to keep the focus on the audiovisual content.

## 🛠️ Tech Stack
* **Framework:** [Next.js]
* **Styling:** [Tailwind CSS]
* **Deployment:** [Cloudflare]

## 💡 Project Motivation
While many creative professionals use template builders (Wix/Squarespace), I chose to build this site from scratch to:
1.  Apply **React Component Architecture** in a real-world production environment.
2.  Solve specific challenges related to **media handling** on the web.
3.  Demonstrate my ability to translate design requirements into clean, maintainable code.
